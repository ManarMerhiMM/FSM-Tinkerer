Welcome to FSM Tinkerer

Choose a tool to start exploring NFAs, DFAs, and DAFSAs.

FSM Tinkerer is a small educational toolkit for experimenting with: nondeterministic finite automata (NFAs), deterministic finite automata (DFAs), and deterministic acyclic finite state automata (DAFSAs).

Use the tools below to convert NFAs to DFAs, build & minimize DAFSAs from a finite language, and read about two practical FSM applications.

DAFSA Builder
Exercise 1

Upload a finite language in JSON format and build its deterministic acyclic finite-state automaton. Then apply minimization to merge equivalent suffix states.

Input: ExampleLanguage.json
Tasks: build DAFSA, minimize, test membership.

Open DAFSA Tool DAFSA.html
NFA → DFA Converter
Exercise 2

Visualize a nondeterministic automaton and automatically convert it to an equivalent deterministic automaton using the subset construction algorithm.

Input: ExampleNFA.json
Tasks: load NFA, convert to DFA, test strings.

Open NFA → DFA Tool NFAtoDFA.html
FSM Applications
Exercise 3

Read through two worked examples of real-world FSMs: a traffic light controller and a vending machine controller, including formal definitions and transition tables.

Content: definitions, tables, diagrams.
Mode: read-only reference.

Open Applications FSM_Applications.html
